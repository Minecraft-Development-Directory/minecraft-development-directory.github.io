Minecraft Development Directory is your gateway to expand your Minecraft experience through custom mods, plugins, data packs, and more. Whether you're a seasoned developer or just starting out, this guide will help you navigate the world of Minecraft development.

On this site, you'll find a wealth of resources, including tutorials, guides, and tools tailored for Minecraft developers. Our goal is to provide you with the knowledge and support you need to create amazing content for the Minecraft community.

Ready to dive in? Explore the sections below to begin your journey into Minecraft development!

::card-group
  :::card{icon="i-lucide-package" title="Mods"}
  Mods add or change features in Minecraft. Start here to learn how to create your own.
  :::

  :::card{icon="i-lucide-plug" title="Plugins"}
  Plugins extend server functionality. Learn to develop plugins for platforms like Spigot and Bukkit.
  :::

  :::card{icon="i-lucide-archive" title="Data Packs"}
  Data packs customize worlds without code changes. Discover how to make and use them.
  :::

  :::card{icon="i-lucide-app-window" title="Launchers"}
  Launchers let you start Minecraft with custom mods or settings. Learn to build your own.
  :::
::

## FAQ

::accordion
  :::accordion-item
  ---
  label: I am new to Minecraft development. Is this site suitable for beginners?
  ---
  Yes, this site is designed to be accessible for beginners. We provide step-by-step guides and resources to help you get started with Minecraft development.
  :::

  :::accordion-item
  ---
  label: What is the difference between mods, plugins, and data packs?
  ---
  Mods add or change features in Minecraft, plugins extend server functionality, and data packs customize game mechanics without code changes. Each serves a unique purpose in enhancing the Minecraft experience.
  :::

  :::accordion-item
  ---
  label: Can I contribute to the Minecraft Development Directory?
  ---
  Yes, we welcome contributions! You can help by submitting tutorials, guides, or tools that you think would benefit the community.
  You can find more information on our [contribution guidelines](./getting-started/contributing).
  :::
